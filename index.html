Что внутри

Полный набор инструментов для диагностики и безопасной оптимизации Rust под Windows.

🛡

Health Check

Score 0-100, проверка RAM, диска, Game Mode, DVR, snapshots, фоновых приложений и безопасный Auto Fix.

🚀

Game Session

Временный профиль, запуск Rust, High priority и авто-откат после выхода из игры.

📈

Stutter Monitor

CPU/RAM/Disk/Rust process telemetry без оверлея и инжекта. Отчёт + CSV.

🧹

Cleaner / Repair

Очистка Rust logs, shader cache, crash dumps, DirectX/GPU cache и Steam Validate Rust.

🗂

Config Manager

Snapshots, restore, export ZIP и recommended settings TXT без опасного изменения cfg.

📋

Report Center

Создаёт HTML/ZIP support bundle со всей диагностикой для удобной отправки.

Безопасность

  • Не внедряется в процесс Rust
  • Не читает и не меняет память игры
  • Не патчит EAC/Rust файлы
  • Нет аккаунтов и телеметрии
  • Все registry-твики логируются и откатываются
  • Перед cfg-операциями есть snapshots

Что меняет

Game Mode: ON
Xbox DVR/background capture: OFF
Power plan: High/Ultimate Performance
Rust GPU preference: High performance
Fullscreen optimizations: optional
Shader/log/cache cleanup: optional

PyInstaller one-file EXE иногда ловит false positives. Можно запускать исходник Python или собрать EXE самостоятельно.

Установка

Самый простой вариант — скачать ZIP из GitHub Releases и запустить от администратора.

Download

Скачай RustFPSOptimizer_v1.0_Pro_Clean.zip из Releases.

Extract

Распакуй архив полностью. Не вытаскивай один .py файл отдельно.

Run

Запусти Run_Pro_Source_As_Admin.bat или собери EXE.

Health Check

Начни с Health Check → Auto Fix Safe Issues.

Build EXE

Build_EXE_Pro_Windows.bat

FAQ

Коротко о главном.

Это чит?

Нет. Это внешняя Windows-утилита. Она не трогает память игры и EAC.

Даст +100 FPS?

Нет магии. Но может убрать причины статтеров: DVR, фон, кэши, power plan, RAM/disk bottleneck.

Можно откатить?

Да. Есть undo, Game Session auto-restore, cfg snapshots и logs.
