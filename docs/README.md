# GitHub Pages site

This folder is a static landing page for Rust FPS Optimizer Pro.

## Project Pages setup

1. Push this repository to GitHub.
2. Go to **Settings → Pages**.
3. Source: **Deploy from a branch**.
4. Branch: `main`, folder: `/docs`.
5. Save.

URL will be:

```text
https://USERNAME.github.io/REPOSITORY/
```

## User site setup

If you want URL like:

```text
https://USERNAME.github.io/
```

create a repository named exactly:

```text
USERNAME.github.io
```

and put `index.html` in the repository root.
