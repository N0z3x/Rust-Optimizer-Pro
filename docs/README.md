# GitHub Pages site

Static landing page for Rust FPS Optimizer Pro.

## Important

`index.html` uses inline SVG icons, so there is no `assets/` folder required.

## Project Pages setup

1. Push this repository to GitHub.
2. Go to **Settings → Pages**.
3. Source: **Deploy from a branch**.
4. Branch: `main`, folder: `/docs`.
5. Save.

URL will be:

```text
https://USERNAME.github.io/REPOSITORY/
```

## Custom domain

This folder includes:

```text
CNAME
```

with:

```text
rustfpsoptimizer.com
```

If you do not want to use this domain, delete `CNAME` before pushing.

For DNS records, see:

```text
DOMAIN_SETUP.md
```
